# Roadmap luyện 200 bài C#

## Giai đoạn 1 - Nền tảng cú pháp

Làm P001 đến P075.

Mục tiêu:

- Đọc input không bị rối.
- Viết được `if`, `for`, `while`.
- Biết xử lý số học cơ bản.

## Giai đoạn 2 - Mảng, List, chuỗi

Làm P076 đến P145.

Mục tiêu:

- Duyệt mảng bằng `for` và `foreach`.
- Tìm kiếm, lọc, sắp xếp.
- Xử lý chuỗi thực tế.

## Giai đoạn 3 - Hàm, đệ quy, thuật toán

Làm P146 đến P165.

Mục tiêu:

- Tách code thành hàm nhỏ.
- Hiểu stack/queue cơ bản.
- Làm quen quy hoạch động đơn giản.

## Giai đoạn 4 - OOP và xử lý dữ liệu

Làm P166 đến P200.

Mục tiêu:

- Tạo class đúng mục đích.
- Gom nhóm dữ liệu.
- Mô phỏng task gần với app quản lý thực tế.

## Sau khi xong 200 bài

Làm tiếp 2 project để đưa vào CV:

1. WinForms Account Manager + lưu file/JSON.
2. ASP.NET Core Web API CRUD + SQL Server.
