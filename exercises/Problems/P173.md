# P173 - Class OrderItem - tổng đơn hàng

**Mức độ:** Trung bình  
**Chủ đề:** OOP  
**Tags:** OOP

---

## Đề bài

Tạo class OrderItem gồm ProductName, Price, Quantity. Tính tổng đơn hàng sau giảm giá %.

## Input

Dòng 1: n. n dòng: name price quantity. Dòng cuối: discountPercent.

## Output

In số tiền phải trả.

## Ví dụ

### Input

```text
2
A 10000 2
B 5000 1
10
```

### Output

```text
22500
```
