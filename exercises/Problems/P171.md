# P171 - Class BankAccount - nạp rút tiền

**Mức độ:** Trung bình  
**Chủ đề:** OOP  
**Tags:** OOP

---

## Đề bài

Tạo class BankAccount có Balance, Deposit, Withdraw. Mô phỏng các giao dịch.

## Input

Dòng 1: balance ban đầu. Dòng 2: q. q dòng: DEPOSIT amount hoặc WITHDRAW amount.

## Output

Sau mỗi lệnh in số dư hoặc Insufficient.

## Ví dụ

### Input

```text
1000
3
DEPOSIT 500
WITHDRAW 300
WITHDRAW 2000
```

### Output

```text
1500
1200
Insufficient
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
