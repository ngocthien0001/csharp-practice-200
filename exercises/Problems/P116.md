# P116 - Độ dài chuỗi

**Mức độ:** Cơ bản  
**Chủ đề:** Chuỗi  
**Tags:** Chuỗi

---

## Đề bài

Nhập một chuỗi. In độ dài chuỗi.

## Input

Một dòng chuỗi.

## Output

In số ký tự của chuỗi.

## Ví dụ

### Input

```text
hello
```

### Output

```text
5
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
