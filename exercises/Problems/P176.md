# P176 - Class Circle - tổng diện tích

**Mức độ:** Trung bình  
**Chủ đề:** OOP  
**Tags:** OOP

---

## Đề bài

Tạo class Circle có Radius và Area(). Tính tổng diện tích các hình tròn với PI = 3.14.

## Input

Dòng 1: n. Dòng 2: n bán kính.

## Output

In tổng diện tích với 2 chữ số.

## Ví dụ

### Input

```text
3
1 2 3
```

### Output

```text
43.96
```

- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
