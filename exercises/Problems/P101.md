# P101 - Đếm cặp có tổng bằng k

**Mức độ:** Trung bình  
**Chủ đề:** Mảng/List  
**Tags:** Mảng/List

---

## Đề bài

Nhập n, k và mảng. Đếm số cặp i < j có a[i] + a[j] = k.

## Input

Dòng 1: n k. Dòng 2: n số.

## Output

In số cặp.

## Ví dụ

### Input

```text
5 5
1 2 3 4 5
```

### Output

```text
2
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
