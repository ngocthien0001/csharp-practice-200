# P155 - Kiểm tra ngoặc hợp lệ

**Mức độ:** Trung bình  
**Chủ đề:** Stack  
**Tags:** Stack

---

## Đề bài

Nhập chuỗi chỉ gồm các loại ngoặc (), [], {}. Kiểm tra chuỗi ngoặc hợp lệ.

## Input

Một chuỗi ngoặc.

## Output

In YES hoặc NO.

## Ví dụ

### Input

```text
([]){}
```

### Output

```text
YES
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
