# P131 - Kiểm tra anagram

**Mức độ:** Trung bình  
**Chủ đề:** Chuỗi  
**Tags:** Chuỗi

---

## Đề bài

Nhập hai chuỗi. Kiểm tra chúng có phải anagram không, bỏ qua khoảng trắng và không phân biệt hoa thường.

## Input

Hai dòng chuỗi.

## Output

In YES hoặc NO.

## Ví dụ

### Input

```text
listen
silent
```

### Output

```text
YES
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
