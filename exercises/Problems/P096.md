# P096 - Xoay phải mảng

**Mức độ:** Trung bình  
**Chủ đề:** Mảng/List  
**Tags:** Mảng/List

---

## Đề bài

Nhập n, k và mảng. Xoay phải mảng k bước.

## Input

Dòng 1: n k. Dòng 2: n số nguyên.

## Output

In mảng sau khi xoay phải.

## Ví dụ

### Input

```text
5 2
1 2 3 4 5
```

### Output

```text
4 5 1 2 3
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
