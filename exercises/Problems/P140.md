# P140 - Nén chuỗi RLE

**Mức độ:** Trung bình  
**Chủ đề:** Chuỗi  
**Tags:** Chuỗi

---

## Đề bài

Nhập chuỗi. Nén theo dạng ký tự + số lần lặp liên tiếp.

## Input

Một dòng chuỗi.

## Output

In chuỗi đã nén.

## Ví dụ

### Input

```text
aaabbc
```

### Output

```text
a3b2c1
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
