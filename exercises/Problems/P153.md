# P153 - Đếm số lần đổi chỗ Bubble Sort

**Mức độ:** Khá  
**Chủ đề:** Sắp xếp  
**Tags:** Sắp xếp

---

## Đề bài

Nhập mảng. Mô phỏng Bubble Sort tăng dần và đếm số lần đổi chỗ.

## Input

Dòng 1: n. Dòng 2: n số.

## Output

In số lần swap.

## Ví dụ

### Input

```text
3
3 2 1
```

### Output

```text
3
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
