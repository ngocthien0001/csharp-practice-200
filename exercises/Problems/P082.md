# P082 - Tổng phần tử chia hết cho k

**Mức độ:** Cơ bản  
**Chủ đề:** Mảng/List  
**Tags:** Mảng/List

---

## Đề bài

Nhập n, k và mảng. Tính tổng các phần tử chia hết cho k.

## Input

Dòng 1: n k. Dòng 2: n số nguyên.

## Output

In tổng.

## Ví dụ

### Input

```text
5 3
1 3 6 7 9
```

### Output

```text
18
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
