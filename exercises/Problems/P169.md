# P169 - Class Account - nhiều gold nhất

**Mức độ:** Trung bình  
**Chủ đề:** OOP  
**Tags:** OOP

---

## Đề bài

Tạo class Account gồm Id, Username, Gold. Tìm account có Gold cao nhất.

## Input

Dòng 1: n. n dòng: id username gold.

## Output

In username và gold.

## Ví dụ

### Input

```text
3
1 acc1 100
2 acc2 500
3 acc3 300
```

### Output

```text
acc2 500
```
