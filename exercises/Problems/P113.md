# P113 - Điểm yên ngựa

**Mức độ:** Khá  
**Chủ đề:** Ma trận  
**Tags:** Ma trận

---

## Đề bài

Điểm yên ngựa là phần tử nhỏ nhất trên dòng và lớn nhất trên cột. Tìm điểm yên ngựa đầu tiên theo thứ tự dòng rồi cột.

## Input

Dòng 1: n m. Sau đó n dòng ma trận.

## Output

In i j value nếu có, ngược lại None. Chỉ số 0-based.

## Ví dụ

### Input

```text
3 3
3 1 3
4 2 5
5 3 6
```

### Output

```text
2 1 3
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
