# P104 - Tổng dãy con liên tiếp lớn nhất

**Mức độ:** Trung bình  
**Chủ đề:** Mảng/List  
**Tags:** Mảng/List

---

## Đề bài

Nhập n và mảng. Tìm tổng lớn nhất của một dãy con liên tiếp không rỗng.

## Input

Dòng 1: n. Dòng 2: n số.

## Output

In tổng lớn nhất.

## Ví dụ

### Input

```text
5
-2 3 -1 5 -6
```

### Output

```text
7
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
