# P125 - Đếm ký tự c

**Mức độ:** Cơ bản  
**Chủ đề:** Chuỗi  
**Tags:** Chuỗi

---

## Đề bài

Nhập chuỗi s và ký tự c. Đếm c xuất hiện trong s.

## Input

Dòng 1: chuỗi s. Dòng 2: ký tự c.

## Output

In số lần xuất hiện.

## Ví dụ

### Input

```text
banana
a
```

### Output

```text
3
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
