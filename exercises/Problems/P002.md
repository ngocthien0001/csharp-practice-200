# P002 - Hiệu hai số nguyên

**Mức độ:** Cơ bản  
**Chủ đề:** Nhập xuất - Toán học  
**Tags:** Nhập xuất - Toán học

---

## Đề bài

Nhập hai số nguyên a, b. Hãy in ra hiệu a - b.

## Input

Một dòng gồm hai số nguyên a và b.

## Output

In ra a - b.

## Ví dụ

### Input

```text
7 4
```

### Output

```text
3
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
