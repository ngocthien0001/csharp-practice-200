# P085 - Đếm số lần xuất hiện x

**Mức độ:** Cơ bản  
**Chủ đề:** Mảng/List  
**Tags:** Mảng/List

---

## Đề bài

Nhập n, mảng và x. Đếm x xuất hiện bao nhiêu lần.

## Input

Dòng 1: n. Dòng 2: n số nguyên. Dòng 3: x.

## Output

In số lần xuất hiện.

## Ví dụ

### Input

```text
5
1 2 3 2 2
2
```

### Output

```text
3
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
