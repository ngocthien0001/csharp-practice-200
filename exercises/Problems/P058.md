# P058 - Tổng chuỗi điều hòa

**Mức độ:** Cơ bản  
**Chủ đề:** Vòng lặp  
**Tags:** Vòng lặp

---

## Đề bài

Nhập n. Tính S = 1 + 1/2 + ... + 1/n.

## Input

Một số nguyên n >= 1.

## Output

In S với 3 chữ số sau dấu phẩy.

## Ví dụ

### Input

```text
2
```

### Output

```text
1.500
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
