# P168 - Class Account - đếm trạng thái

**Mức độ:** Trung bình  
**Chủ đề:** OOP  
**Tags:** OOP

---

## Đề bài

Tạo class Account gồm Id, Username, Status. Đếm số tài khoản theo từng trạng thái.

## Input

Dòng 1: n. n dòng: id username status.

## Output

Mỗi dòng: status count, sắp xếp theo status.

## Ví dụ

### Input

```text
5
1 acc1 Running
2 acc2 Die
3 acc3 Running
4 acc4 Online
5 acc5 Die
```

### Output

```text
Die 2
Online 1
Running 2
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
