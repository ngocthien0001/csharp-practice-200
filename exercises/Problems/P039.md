# P039 - Thứ trong tuần

**Mức độ:** Cơ bản  
**Chủ đề:** Switch/Điều kiện  
**Tags:** Switch/Điều kiện

---

## Đề bài

Nhập số từ 1 đến 7. In thứ tương ứng: Monday, Tuesday, ..., Sunday. Nếu sai in Invalid.

## Input

Một số nguyên d.

## Output

In tên thứ.

## Ví dụ

### Input

```text
1
```

### Output

```text
Monday
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
