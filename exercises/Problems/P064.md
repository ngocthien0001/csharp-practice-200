# P064 - Đổi thập phân sang nhị phân

**Mức độ:** Cơ bản  
**Chủ đề:** Vòng lặp  
**Tags:** Vòng lặp

---

## Đề bài

Nhập n không âm. Đổi n sang hệ nhị phân.

## Input

Một số nguyên không âm n.

## Output

In biểu diễn nhị phân của n.

## Ví dụ

### Input

```text
10
```

### Output

```text
1010
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
