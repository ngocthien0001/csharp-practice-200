# P114 - Dòng Pascal

**Mức độ:** Trung bình  
**Chủ đề:** Toán học  
**Tags:** Toán học

---

## Đề bài

Nhập n. In dòng thứ n của tam giác Pascal, tính từ dòng 0.

## Input

Một số nguyên n.

## Output

In các số trên dòng n.

## Ví dụ

### Input

```text
0
```

### Output

```text
1
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
