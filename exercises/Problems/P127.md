# P127 - Kiểm tra chứa chuỗi con

**Mức độ:** Cơ bản  
**Chủ đề:** Chuỗi  
**Tags:** Chuỗi

---

## Đề bài

Nhập chuỗi s và chuỗi sub. Kiểm tra sub có nằm trong s không.

## Input

Dòng 1: s. Dòng 2: sub.

## Output

In YES hoặc NO.

## Ví dụ

### Input

```text
hello world
world
```

### Output

```text
YES
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
