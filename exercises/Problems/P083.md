# P083 - Đảo mảng

**Mức độ:** Cơ bản  
**Chủ đề:** Mảng/List  
**Tags:** Mảng/List

---

## Đề bài

Nhập n và mảng. In mảng theo thứ tự ngược lại.

## Input

Dòng 1: n. Dòng 2: n số nguyên.

## Output

In mảng đảo ngược.

## Ví dụ

### Input

```text
5
1 2 3 4 5
```

### Output

```text
5 4 3 2 1
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
