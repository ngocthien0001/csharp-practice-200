# P006 - Hình chữ nhật

**Mức độ:** Cơ bản  
**Chủ đề:** Toán học  
**Tags:** Toán học

---

## Đề bài

Nhập chiều dài a và chiều rộng b. Tính chu vi và diện tích hình chữ nhật.

## Input

Một dòng gồm hai số nguyên a, b.

## Output

In hai dòng: chu vi, diện tích.

## Ví dụ

### Input

```text
5 3
```

### Output

```text
16
15
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
