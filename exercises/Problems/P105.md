# P105 - Tổng ma trận

**Mức độ:** Cơ bản  
**Chủ đề:** Ma trận  
**Tags:** Ma trận

---

## Đề bài

Nhập ma trận n x m. Tính tổng tất cả phần tử.

## Input

Dòng 1: n m. Sau đó n dòng, mỗi dòng m số.

## Output

In tổng.

## Ví dụ

### Input

```text
2 3
1 2 3
4 5 6
```

### Output

```text
21
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
