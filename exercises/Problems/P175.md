# P175 - Class Rectangle - diện tích lớn nhất

**Mức độ:** Trung bình  
**Chủ đề:** OOP  
**Tags:** OOP

---

## Đề bài

Tạo class Rectangle có Width, Height, Area(). Tìm hình chữ nhật có diện tích lớn nhất.

## Input

Dòng 1: n. n dòng: name width height.

## Output

In name và area.

## Ví dụ

### Input

```text
3
A 2 3
B 4 5
C 10 1
```

### Output

```text
B 20
```
