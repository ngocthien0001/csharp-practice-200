# P033 - Loại tam giác

**Mức độ:** Cơ bản  
**Chủ đề:** Điều kiện  
**Tags:** Điều kiện

---

## Đề bài

Nhập ba cạnh a, b, c. Nếu không phải tam giác in Invalid. Nếu đều in Equilateral, cân in Isosceles, còn lại Scalene.

## Input

Một dòng gồm ba số nguyên dương.

## Output

In loại tam giác.

## Ví dụ

### Input

```text
3 3 3
```

### Output

```text
Equilateral
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
