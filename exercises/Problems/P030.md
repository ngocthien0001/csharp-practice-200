# P030 - Giá trị tuyệt đối

**Mức độ:** Cơ bản  
**Chủ đề:** Điều kiện  
**Tags:** Điều kiện

---

## Đề bài

Nhập số nguyên n. In giá trị tuyệt đối của n.

## Input

Một số nguyên n.

## Output

In |n|.

## Ví dụ

### Input

```text
-5
```

### Output

```text
5
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
