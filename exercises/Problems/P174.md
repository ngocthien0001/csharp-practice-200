# P174 - Class Car - lọc xe vượt tốc độ

**Mức độ:** Trung bình  
**Chủ đề:** OOP  
**Tags:** OOP

---

## Đề bài

Tạo class Car gồm Plate và Speed. In biển số xe có tốc độ vượt giới hạn.

## Input

Dòng 1: n. Dòng 2: speedLimit. n dòng: plate speed.

## Output

In các plate vượt tốc độ hoặc None.

## Ví dụ

### Input

```text
3
60
A 50
B 70
C 80
```

### Output

```text
B C
```
