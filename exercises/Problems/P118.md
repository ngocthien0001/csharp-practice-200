# P118 - Đếm chữ số chữ cái khoảng trắng

**Mức độ:** Cơ bản  
**Chủ đề:** Chuỗi  
**Tags:** Chuỗi

---

## Đề bài

Nhập một chuỗi. Đếm số chữ cái, chữ số và khoảng trắng.

## Input

Một dòng chuỗi.

## Output

In ba số: letters digits spaces.

## Ví dụ

### Input

```text
abc 123
```

### Output

```text
3 3 1
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
