# P177 - Class Point - khoảng cách xa nhất

**Mức độ:** Khá  
**Chủ đề:** OOP  
**Tags:** OOP

---

## Đề bài

Tạo class Point có X, Y và phương thức DistanceTo(). Tìm khoảng cách lớn nhất giữa hai điểm.

## Input

Dòng 1: n. n dòng: x y.

## Output

In khoảng cách lớn nhất với 2 chữ số.

## Ví dụ

### Input

```text
3
0 0
3 4
1 1
```

### Output

```text
5.00
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
