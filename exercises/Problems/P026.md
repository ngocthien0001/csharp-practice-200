# P026 - Âm dương hay bằng 0

**Mức độ:** Cơ bản  
**Chủ đề:** Điều kiện  
**Tags:** Điều kiện

---

## Đề bài

Nhập số nguyên n. Xác định n là số dương, âm hay bằng 0.

## Input

Một số nguyên n.

## Output

In Positive, Negative hoặc Zero.

## Ví dụ

### Input

```text
5
```

### Output

```text
Positive
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
