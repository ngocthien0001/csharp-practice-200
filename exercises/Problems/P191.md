# P191 - Khoảng cách ngày

**Mức độ:** Trung bình  
**Chủ đề:** DateTime  
**Tags:** DateTime

---

## Đề bài

Nhập hai ngày dạng yyyy-MM-dd. Tính số ngày chênh lệch tuyệt đối.

## Input

Hai dòng ngày yyyy-MM-dd.

## Output

In số ngày chênh lệch.

## Ví dụ

### Input

```text
2025-01-01
2025-01-31
```

### Output

```text
30
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
