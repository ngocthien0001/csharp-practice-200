# P043 - Tổng từ 1 đến n

**Mức độ:** Cơ bản  
**Chủ đề:** Vòng lặp  
**Tags:** Vòng lặp

---

## Đề bài

Nhập n. Tính tổng 1 + 2 + ... + n.

## Input

Một số nguyên n >= 1.

## Output

In tổng.

## Ví dụ

### Input

```text
5
```

### Output

```text
15
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
