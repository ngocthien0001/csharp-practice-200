# P036 - Tính thuế thu nhập đơn giản

**Mức độ:** Cơ bản  
**Chủ đề:** Điều kiện  
**Tags:** Điều kiện

---

## Đề bài

Nhập lương. Nếu lương <= 10000000 thuế 0%, <= 20000000 thuế 10% phần vượt 10000000, còn lại thuế 10% cho 10000000 tiếp theo và 20% phần vượt 20000000.

## Input

Một số nguyên salary.

## Output

In số tiền thuế.

## Ví dụ

### Input

```text
8000000
```

### Output

```text
0
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
