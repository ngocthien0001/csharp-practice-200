# P075 - Số bước Collatz

**Mức độ:** Trung bình  
**Chủ đề:** Vòng lặp  
**Tags:** Vòng lặp

---

## Đề bài

Cho n. Nếu n chẵn chia 2, nếu n lẻ nhân 3 cộng 1. Đếm số bước để về 1.

## Input

Một số nguyên n > 0.

## Output

In số bước.

## Ví dụ

### Input

```text
1
```

### Output

```text
0
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
