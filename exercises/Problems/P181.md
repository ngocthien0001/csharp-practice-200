# P181 - Class Contact - tìm kiếm danh bạ

**Mức độ:** Trung bình  
**Chủ đề:** OOP  
**Tags:** OOP

---

## Đề bài

Tạo class Contact gồm Name, Phone. Tìm các liên hệ chứa keyword trong tên hoặc số điện thoại.

## Input

Dòng 1: n. n dòng: name phone. Dòng cuối: keyword.

## Output

In các contact khớp hoặc None.

## Ví dụ

### Input

```text
3
An 0901
Binh 0818
Chi 0777
08
```

### Output

```text
Binh 0818
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
