# P170 - Class Employee - tổng và trung bình lương

**Mức độ:** Trung bình  
**Chủ đề:** OOP  
**Tags:** OOP

---

## Đề bài

Tạo class Employee gồm Id, Name, Salary. Tính tổng lương và lương trung bình.

## Input

Dòng 1: n. n dòng: id name salary.

## Output

In total và average với 2 chữ số.

## Ví dụ

### Input

```text
3
1 An 1000
2 Binh 2000
3 Chi 3000
```

### Output

```text
6000 2000.00
```
