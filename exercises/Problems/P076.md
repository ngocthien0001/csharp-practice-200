# P076 - Tổng mảng

**Mức độ:** Cơ bản  
**Chủ đề:** Mảng/List  
**Tags:** Mảng/List

---

## Đề bài

Nhập n và mảng n số nguyên. Tính tổng các phần tử.

## Input

Dòng 1: n. Dòng 2: n số nguyên.

## Output

In tổng.

## Ví dụ

### Input

```text
5
1 2 3 4 5
```

### Output

```text
15
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
