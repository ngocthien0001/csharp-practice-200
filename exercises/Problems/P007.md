# P007 - Đổi độ C sang độ F

**Mức độ:** Cơ bản  
**Chủ đề:** Toán học  
**Tags:** Toán học

---

## Đề bài

Nhập nhiệt độ C. Đổi sang độ F theo công thức F = C * 9 / 5 + 32.

## Input

Một số thực C.

## Output

In ra F với 1 chữ số sau dấu phẩy.

## Ví dụ

### Input

```text
0
```

### Output

```text
32.0
```
