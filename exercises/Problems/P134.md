# P134 - Mã hóa Caesar

**Mức độ:** Trung bình  
**Chủ đề:** Chuỗi  
**Tags:** Chuỗi

---

## Đề bài

Nhập chuỗi và số k. Dịch mỗi chữ cái đi k vị trí trong bảng chữ cái. Ký tự khác giữ nguyên.

## Input

Dòng 1: chuỗi. Dòng 2: k.

## Output

In chuỗi sau khi mã hóa.

## Ví dụ

### Input

```text
abcXYZ
2
```

### Output

```text
cdeZAB
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
