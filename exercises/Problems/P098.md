# P098 - Xóa theo vị trí

**Mức độ:** Trung bình  
**Chủ đề:** Mảng/List  
**Tags:** Mảng/List

---

## Đề bài

Nhập n, mảng và index. Xóa phần tử tại index. Nếu index không hợp lệ giữ nguyên mảng.

## Input

Dòng 1: n. Dòng 2: n số. Dòng 3: index.

## Output

In mảng sau khi xóa.

## Ví dụ

### Input

```text
4
1 2 3 4
1
```

### Output

```text
1 3 4
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
