# P154 - Selection Sort sau p lượt

**Mức độ:** Khá  
**Chủ đề:** Sắp xếp  
**Tags:** Sắp xếp

---

## Đề bài

Nhập mảng và số p. Thực hiện p lượt đầu của Selection Sort tăng dần.

## Input

Dòng 1: n. Dòng 2: n số. Dòng 3: p.

## Output

In mảng sau p lượt.

## Ví dụ

### Input

```text
5
5 4 3 2 1
2
```

### Output

```text
1 2 3 4 5
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
