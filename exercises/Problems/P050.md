# P050 - Số đối xứng

**Mức độ:** Cơ bản  
**Chủ đề:** Vòng lặp  
**Tags:** Vòng lặp

---

## Đề bài

Nhập n. Kiểm tra n có phải số đối xứng không.

## Input

Một số nguyên không âm n.

## Output

In YES hoặc NO.

## Ví dụ

### Input

```text
121
```

### Output

```text
YES
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
