# P167 - Class Product - tổng giá trị kho

**Mức độ:** Trung bình  
**Chủ đề:** OOP  
**Tags:** OOP

---

## Đề bài

Tạo class Product gồm Code, Price, Quantity. Tính tổng giá trị kho.

## Input

Dòng 1: n. n dòng: code price quantity.

## Output

In tổng price * quantity.

## Ví dụ

### Input

```text
3
A01 10000 2
B02 5000 5
C03 20000 1
```

### Output

```text
65000
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
