# P158 - Độ sâu ngoặc tròn

**Mức độ:** Trung bình  
**Chủ đề:** Stack  
**Tags:** Stack

---

## Đề bài

Nhập chuỗi ngoặc tròn hợp lệ. Tìm độ sâu lớn nhất.

## Input

Một chuỗi ngoặc tròn hợp lệ.

## Output

In độ sâu lớn nhất.

## Ví dụ

### Input

```text
((()))
```

### Output

```text
3
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
