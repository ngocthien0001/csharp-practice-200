# P080 - Đếm dương âm không

**Mức độ:** Cơ bản  
**Chủ đề:** Mảng/List  
**Tags:** Mảng/List

---

## Đề bài

Nhập n và mảng. Đếm số dương, âm, bằng 0.

## Input

Dòng 1: n. Dòng 2: n số nguyên.

## Output

In ba số: positive negative zero.

## Ví dụ

### Input

```text
5
-1 0 2 3 0
```

### Output

```text
2 1 2
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
