# P032 - Kiểm tra tam giác

**Mức độ:** Cơ bản  
**Chủ đề:** Điều kiện  
**Tags:** Điều kiện

---

## Đề bài

Nhập ba cạnh a, b, c. Kiểm tra có tạo thành tam giác không.

## Input

Một dòng gồm ba số nguyên dương.

## Output

In YES nếu là tam giác, ngược lại NO.

## Ví dụ

### Input

```text
3 4 5
```

### Output

```text
YES
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
