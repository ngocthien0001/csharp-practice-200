# P088 - Sắp xếp mảng tăng dần

**Mức độ:** Cơ bản  
**Chủ đề:** Sắp xếp  
**Tags:** Sắp xếp

---

## Đề bài

Nhập n và mảng. In mảng sau khi sắp xếp tăng dần.

## Input

Dòng 1: n. Dòng 2: n số nguyên.

## Output

In mảng tăng dần.

## Ví dụ

### Input

```text
5
3 1 4 1 2
```

### Output

```text
1 1 2 3 4
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
