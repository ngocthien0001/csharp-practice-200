# P179 - Class CourseResult - báo cáo điểm

**Mức độ:** Trung bình  
**Chủ đề:** OOP  
**Tags:** OOP

---

## Đề bài

Tạo class CourseResult gồm StudentName, Score. Đếm số sinh viên qua môn và điểm trung bình.

## Input

Dòng 1: n. n dòng: name score.

## Output

In passCount và average.

## Ví dụ

### Input

```text
3
An 8
Binh 4
Chi 6
```

### Output

```text
2 6.00
```
