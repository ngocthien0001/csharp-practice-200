# P144 - Chuẩn hóa họ tên

**Mức độ:** Cơ bản  
**Chủ đề:** Chuỗi  
**Tags:** Chuỗi

---

## Đề bài

Nhập họ tên có thể thừa khoảng trắng và sai hoa thường. Chuẩn hóa mỗi từ viết hoa chữ đầu.

## Input

Một dòng họ tên.

## Output

In họ tên đã chuẩn hóa.

## Ví dụ

### Input

```text
  do   NGOC thien 
```

### Output

```text
Do Ngoc Thien
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
