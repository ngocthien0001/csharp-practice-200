# P119 - Đảo chuỗi

**Mức độ:** Cơ bản  
**Chủ đề:** Chuỗi  
**Tags:** Chuỗi

---

## Đề bài

Nhập một chuỗi. In chuỗi đảo ngược.

## Input

Một dòng chuỗi.

## Output

In chuỗi đảo ngược.

## Ví dụ

### Input

```text
hello
```

### Output

```text
olleh
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
