# P126 - Thay thế ký tự

**Mức độ:** Cơ bản  
**Chủ đề:** Chuỗi  
**Tags:** Chuỗi

---

## Đề bài

Nhập chuỗi s, ký tự old và ký tự new. Thay mọi old bằng new.

## Input

Dòng 1: chuỗi s. Dòng 2: old. Dòng 3: new.

## Output

In chuỗi sau khi thay.

## Ví dụ

### Input

```text
banana
a
o
```

### Output

```text
bonono
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
