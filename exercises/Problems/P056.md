# P056 - Số Fibonacci thứ n

**Mức độ:** Cơ bản  
**Chủ đề:** Vòng lặp  
**Tags:** Vòng lặp

---

## Đề bài

Với F0 = 0, F1 = 1. Nhập n, in Fn.

## Input

Một số nguyên n từ 0 đến 40.

## Output

In Fn.

## Ví dụ

### Input

```text
0
```

### Output

```text
0
```
