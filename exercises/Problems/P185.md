# P185 - Class LogEntry - lọc log

**Mức độ:** Trung bình  
**Chủ đề:** OOP  
**Tags:** OOP

---

## Đề bài

Nhập nhiều dòng log và keyword. In các log chứa keyword, không phân biệt hoa thường.

## Input

Dòng 1: n. Dòng 2: keyword. n dòng tiếp: log text.

## Output

In các dòng log khớp hoặc None.

## Ví dụ

### Input

```text
3
error
[INFO] Start
[ERROR] Fail
[WARN] Low
```

### Output

```text
[ERROR] Fail
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
