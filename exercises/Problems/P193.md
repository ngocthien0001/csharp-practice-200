# P193 - Tổng doanh thu theo tháng

**Mức độ:** Trung bình  
**Chủ đề:** LINQ/Data  
**Tags:** LINQ/Data

---

## Đề bài

Nhập các giao dịch gồm ngày yyyy-MM-dd và amount. Tính tổng amount theo tháng yyyy-MM.

## Input

Dòng 1: n. n dòng: date amount.

## Output

Mỗi dòng: yyyy-MM total.

## Ví dụ

### Input

```text
3
2025-01-01 100
2025-01-20 50
2025-02-01 200
```

### Output

```text
2025-01 150
2025-02 200
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
