# P146 - Viết hàm tính min max

**Mức độ:** Cơ bản  
**Chủ đề:** Hàm  
**Tags:** Hàm

---

## Đề bài

Nhập n và mảng. Hãy tách logic tìm min max thành hàm riêng.

## Input

Dòng 1: n. Dòng 2: n số nguyên.

## Output

In min và max.

## Ví dụ

### Input

```text
5
1 9 -2 3 0
```

### Output

```text
-2 9
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
