# P068 - Số 0 tận cùng của n!

**Mức độ:** Cơ bản  
**Chủ đề:** Vòng lặp  
**Tags:** Vòng lặp

---

## Đề bài

Nhập n. Đếm số chữ số 0 ở cuối n!.

## Input

Một số nguyên n >= 0.

## Output

In số lượng chữ số 0 tận cùng.

## Ví dụ

### Input

```text
5
```

### Output

```text
1
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
