# P186 - CSV tổng cột amount

**Mức độ:** Trung bình  
**Chủ đề:** Xử lý dữ liệu  
**Tags:** Xử lý dữ liệu

---

## Đề bài

Nhập n dòng CSV dạng name,amount. Tính tổng amount.

## Input

Dòng 1: n. n dòng: name,amount.

## Output

In tổng amount.

## Ví dụ

### Input

```text
3
A,100
B,200
C,50
```

### Output

```text
350
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
