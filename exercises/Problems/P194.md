# P194 - Tỉ lệ qua môn theo lớp

**Mức độ:** Trung bình  
**Chủ đề:** LINQ/Data  
**Tags:** LINQ/Data

---

## Đề bài

Nhập sinh viên gồm name, class, score. Đếm số qua môn / tổng theo từng lớp.

## Input

Dòng 1: n. n dòng: name class score.

## Output

Mỗi dòng: class pass/total, sắp xếp theo class.

## Ví dụ

### Input

```text
4
An A 8
Binh A 4
Chi B 6
Dung B 3
```

### Output

```text
A 1/2
B 1/2
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
