# P071 - Hình chữ nhật dấu sao

**Mức độ:** Cơ bản  
**Chủ đề:** Vòng lặp lồng nhau  
**Tags:** Vòng lặp lồng nhau

---

## Đề bài

Nhập h và w. In hình chữ nhật h dòng, mỗi dòng w dấu sao.

## Input

Một dòng gồm h và w.

## Output

In hình chữ nhật dấu sao.

## Ví dụ

### Input

```text
3 4
```

### Output

```text
****
****
****
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
