# P180 - Class TodoItem - quản lý việc cần làm

**Mức độ:** Trung bình  
**Chủ đề:** OOP  
**Tags:** OOP

---

## Đề bài

Mô phỏng Todo list. ADD text thêm việc, DONE index xóa việc theo index 0-based, LIST in danh sách.

## Input

Dòng 1: q. q dòng lệnh.

## Output

In kết quả sau mỗi lệnh.

## Ví dụ

### Input

```text
5
ADD learn C#
ADD build app
LIST
DONE 0
LIST
```

### Output

```text
1
2
learn C#|build app
OK
build app
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
