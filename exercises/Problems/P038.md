# P038 - Giảm giá đơn hàng

**Mức độ:** Cơ bản  
**Chủ đề:** Điều kiện  
**Tags:** Điều kiện

---

## Đề bài

Nhập tổng tiền đơn hàng. Nếu >= 1000000 giảm 10%, nếu >= 500000 giảm 5%, còn lại không giảm.

## Input

Một số nguyên total.

## Output

In số tiền phải trả sau giảm giá.

## Ví dụ

### Input

```text
1200000
```

### Output

```text
1080000
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
