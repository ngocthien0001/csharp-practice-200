# P195 - Sản phẩm sắp hết hàng

**Mức độ:** Trung bình  
**Chủ đề:** LINQ/Data  
**Tags:** LINQ/Data

---

## Đề bài

Nhập danh sách sản phẩm và ngưỡng tồn kho. In sản phẩm có quantity < threshold.

## Input

Dòng 1: n. Dòng 2: threshold. n dòng: name quantity.

## Output

In tên sản phẩm hoặc None.

## Ví dụ

### Input

```text
3
5
A 2
B 5
C 0
```

### Output

```text
A C
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
