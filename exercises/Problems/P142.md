# P142 - Chuỗi số nguyên

**Mức độ:** Cơ bản  
**Chủ đề:** Chuỗi  
**Tags:** Chuỗi

---

## Đề bài

Nhập một chuỗi. Kiểm tra chuỗi có phải số nguyên hợp lệ không. Cho phép dấu - ở đầu.

## Input

Một dòng chuỗi.

## Output

In YES hoặc NO.

## Ví dụ

### Input

```text
123
```

### Output

```text
YES
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
