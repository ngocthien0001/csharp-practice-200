# P164 - Balo 0/1

**Mức độ:** Khá  
**Chủ đề:** Quy hoạch động  
**Tags:** Quy hoạch động

---

## Đề bài

Có n món đồ, mỗi món có weight và value. Chọn mỗi món tối đa một lần để tổng weight <= W và value lớn nhất.

## Input

Dòng 1: n W. n dòng tiếp theo: weight value.

## Output

In value lớn nhất.

## Ví dụ

### Input

```text
3 5
2 3
3 4
4 5
```

### Output

```text
7
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
