# P150 - Đệ quy UCLN

**Mức độ:** Cơ bản  
**Chủ đề:** Đệ quy  
**Tags:** Đệ quy

---

## Đề bài

Nhập a, b. Tính UCLN bằng thuật toán Euclid.

## Input

Một dòng gồm hai số nguyên.

## Output

In UCLN.

## Ví dụ

### Input

```text
12 18
```

### Output

```text
6
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
