# P182 - Class CartItem - gom số lượng

**Mức độ:** Trung bình  
**Chủ đề:** OOP  
**Tags:** OOP

---

## Đề bài

Giỏ hàng có nhiều dòng sản phẩm trùng tên. Gom tổng số lượng theo sản phẩm.

## Input

Dòng 1: n. n dòng: product quantity.

## Output

Mỗi dòng: product totalQuantity, sắp xếp theo product.

## Ví dụ

### Input

```text
4
A 2
B 1
A 3
C 5
```

### Output

```text
A 5
B 1
C 5
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
