# P184 - Class Movie - phim điểm cao nhất

**Mức độ:** Trung bình  
**Chủ đề:** OOP  
**Tags:** OOP

---

## Đề bài

Tạo class Movie gồm Title, Rating. Tìm phim rating cao nhất và rating trung bình.

## Input

Dòng 1: n. n dòng: title rating.

## Output

In titleTop ratingTop average.

## Ví dụ

### Input

```text
3
A 8.5
B 9.0
C 7.5
```

### Output

```text
B 9.0 8.33
```
