# P005 - Diện tích hình tròn

**Mức độ:** Cơ bản  
**Chủ đề:** Toán học  
**Tags:** Toán học

---

## Đề bài

Nhập bán kính r. Tính diện tích hình tròn với PI = 3.14.

## Input

Một số thực r.

## Output

In diện tích, làm tròn 2 chữ số sau dấu phẩy.

## Ví dụ

### Input

```text
1.5
```

### Output

```text
7.07
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
