# P067 - Xóa số 0 ở cuối

**Mức độ:** Cơ bản  
**Chủ đề:** Vòng lặp  
**Tags:** Vòng lặp

---

## Đề bài

Nhập số nguyên không âm n. Xóa các chữ số 0 ở cuối n.

## Input

Một số nguyên không âm n.

## Output

In số sau khi xóa 0 ở cuối. Nếu n = 0 thì in 0.

## Ví dụ

### Input

```text
12000
```

### Output

```text
12
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
