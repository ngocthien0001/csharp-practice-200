# P159 - Tính biểu thức hậu tố

**Mức độ:** Khá  
**Chủ đề:** Stack  
**Tags:** Stack

---

## Đề bài

Nhập biểu thức hậu tố gồm số nguyên và toán tử + - * /. Các token cách nhau bởi dấu cách. Phép chia là chia nguyên.

## Input

Một dòng biểu thức hậu tố.

## Output

In kết quả.

## Ví dụ

### Input

```text
2 3 +
```

### Output

```text
5
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
