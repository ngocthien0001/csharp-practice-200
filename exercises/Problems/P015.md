# P015 - Lãi đơn

**Mức độ:** Cơ bản  
**Chủ đề:** Toán học  
**Tags:** Toán học

---

## Đề bài

Nhập số tiền gốc p, lãi suất năm r (%), số năm y. Tính tiền lãi đơn.

## Input

Một dòng gồm p r y.

## Output

In tiền lãi với 2 chữ số sau dấu phẩy.

## Ví dụ

### Input

```text
1000000 5 2
```

### Output

```text
100000.00
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
