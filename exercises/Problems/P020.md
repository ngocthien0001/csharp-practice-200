# P020 - Số lon sơn cần mua

**Mức độ:** Cơ bản  
**Chủ đề:** Toán học  
**Tags:** Toán học

---

## Đề bài

Một lon sơn phủ được 10 mét vuông. Nhập diện tích cần sơn, tính số lon cần mua.

## Input

Một số thực area.

## Output

In số lon sơn cần mua.

## Ví dụ

### Input

```text
25
```

### Output

```text
3
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
