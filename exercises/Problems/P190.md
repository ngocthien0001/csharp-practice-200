# P190 - Đếm level log

**Mức độ:** Trung bình  
**Chủ đề:** Xử lý dữ liệu  
**Tags:** Xử lý dữ liệu

---

## Đề bài

Mỗi dòng log bắt đầu bằng [LEVEL]. Đếm số log theo LEVEL.

## Input

Dòng 1: n. n dòng log.

## Output

Mỗi dòng: LEVEL count, sắp xếp theo LEVEL.

## Ví dụ

### Input

```text
4
[INFO] start
[ERROR] fail
[INFO] ok
[WARN] low
```

### Output

```text
ERROR 1
INFO 2
WARN 1
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
