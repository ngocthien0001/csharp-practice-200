# P001 - Tổng hai số nguyên

**Mức độ:** Cơ bản  
**Chủ đề:** Nhập xuất - Toán học  
**Tags:** input, math

---

## Đề bài

Nhập hai số nguyên a, b. Hãy in ra tổng của chúng.

## Input

Một dòng gồm hai số nguyên a và b.

## Output

In ra a + b.

## Ví dụ

### Input

```text
2 3
```

### Output

```text
5
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
