# P090 - Số lớn thứ hai phân biệt

**Mức độ:** Trung bình  
**Chủ đề:** Mảng/List  
**Tags:** Mảng/List

---

## Đề bài

Nhập n và mảng. Tìm số lớn thứ hai phân biệt. Nếu không tồn tại in None.

## Input

Dòng 1: n. Dòng 2: n số nguyên.

## Output

In số lớn thứ hai hoặc None.

## Ví dụ

### Input

```text
5
1 5 3 5 2
```

### Output

```text
3
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
