# P161 - Lũy thừa modulo

**Mức độ:** Khá  
**Chủ đề:** Thuật toán  
**Tags:** Thuật toán

---

## Đề bài

Nhập a, b, m. Tính a^b mod m bằng lũy thừa nhanh.

## Input

Một dòng gồm a b m.

## Output

In kết quả.

## Ví dụ

### Input

```text
2 10 1000
```

### Output

```text
24
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
