# P069 - Đếm chữ số d

**Mức độ:** Cơ bản  
**Chủ đề:** Vòng lặp  
**Tags:** Vòng lặp

---

## Đề bài

Nhập n và chữ số d. Đếm d xuất hiện bao nhiêu lần trong |n|.

## Input

Một dòng gồm n và d.

## Output

In số lần xuất hiện.

## Ví dụ

### Input

```text
12021 2
```

### Output

```text
2
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
