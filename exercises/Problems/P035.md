# P035 - Máy tính đơn giản

**Mức độ:** Cơ bản  
**Chủ đề:** Điều kiện  
**Tags:** Điều kiện

---

## Đề bài

Nhập a, phép toán op và b. op là +, -, *, /. Với phép chia, in kết quả 2 chữ số sau dấu phẩy.

## Input

Một dòng dạng: a op b.

## Output

In kết quả phép tính.

## Ví dụ

### Input

```text
5 + 3
```

### Output

```text
8
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
