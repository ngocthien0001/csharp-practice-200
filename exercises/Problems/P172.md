# P172 - Class Book - mượn trả sách

**Mức độ:** Trung bình  
**Chủ đề:** OOP  
**Tags:** OOP

---

## Đề bài

Quản lý sách theo mã và số lượng. Lệnh BORROW giảm số lượng nếu còn, RETURN tăng số lượng.

## Input

Dòng 1: n. n dòng: code quantity. Dòng tiếp: q. q dòng lệnh.

## Output

Mỗi lệnh in OK hoặc NO.

## Ví dụ

### Input

```text
2
B1 1
B2 0
4
BORROW B1
BORROW B1
RETURN B2
BORROW B2
```

### Output

```text
OK
NO
OK
OK
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
