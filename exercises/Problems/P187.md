# P187 - Group nhân viên theo phòng ban

**Mức độ:** Trung bình  
**Chủ đề:** LINQ/Data  
**Tags:** LINQ/Data

---

## Đề bài

Nhập danh sách nhân viên gồm name và department. Đếm số nhân viên theo department.

## Input

Dòng 1: n. n dòng: name department.

## Output

Mỗi dòng: department count, sắp xếp theo department.

## Ví dụ

### Input

```text
4
An IT
Binh HR
Chi IT
Dung Sale
```

### Output

```text
HR 1
IT 2
Sale 1
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
