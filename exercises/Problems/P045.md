# P045 - Giai thừa

**Mức độ:** Cơ bản  
**Chủ đề:** Vòng lặp  
**Tags:** Vòng lặp

---

## Đề bài

Nhập n. Tính n!.

## Input

Một số nguyên n từ 0 đến 12.

## Output

In n!.

## Ví dụ

### Input

```text
5
```

### Output

```text
120
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
