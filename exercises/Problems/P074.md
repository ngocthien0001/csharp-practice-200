# P074 - Phân tích thừa số nguyên tố

**Mức độ:** Trung bình  
**Chủ đề:** Vòng lặp  
**Tags:** Vòng lặp

---

## Đề bài

Nhập n > 1. Phân tích n thành thừa số nguyên tố.

## Input

Một số nguyên n > 1.

## Output

In dạng 2^a * 3^b ... Nếu số mũ bằng 1 thì chỉ in số nguyên tố.

## Ví dụ

### Input

```text
12
```

### Output

```text
2^2 * 3
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
