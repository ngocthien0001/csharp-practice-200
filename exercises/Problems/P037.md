# P037 - Đăng nhập đơn giản

**Mức độ:** Cơ bản  
**Chủ đề:** Chuỗi - Điều kiện  
**Tags:** Chuỗi - Điều kiện

---

## Đề bài

Nhập username và password. Tài khoản đúng là admin / 123456.

## Input

Hai dòng: username và password.

## Output

In Login success hoặc Login failed.

## Ví dụ

### Input

```text
admin
123456
```

### Output

```text
Login success
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
