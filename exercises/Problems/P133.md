# P133 - Xóa ký tự trùng

**Mức độ:** Trung bình  
**Chủ đề:** Chuỗi  
**Tags:** Chuỗi

---

## Đề bài

Nhập một chuỗi. Xóa ký tự trùng, giữ lần xuất hiện đầu tiên.

## Input

Một dòng chuỗi.

## Output

In chuỗi sau khi xóa trùng.

## Ví dụ

### Input

```text
banana
```

### Output

```text
ban
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
