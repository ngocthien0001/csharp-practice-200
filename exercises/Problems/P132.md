# P132 - Sắp xếp ký tự

**Mức độ:** Cơ bản  
**Chủ đề:** Chuỗi  
**Tags:** Chuỗi

---

## Đề bài

Nhập một chuỗi. Sắp xếp các ký tự theo mã Unicode tăng dần.

## Input

Một dòng chuỗi.

## Output

In chuỗi sau khi sắp xếp.

## Ví dụ

### Input

```text
cba
```

### Output

```text
abc
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
