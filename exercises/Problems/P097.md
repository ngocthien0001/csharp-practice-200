# P097 - Chèn phần tử

**Mức độ:** Trung bình  
**Chủ đề:** Mảng/List  
**Tags:** Mảng/List

---

## Đề bài

Nhập n, mảng, vị trí index và giá trị x. Chèn x vào index.

## Input

Dòng 1: n. Dòng 2: n số. Dòng 3: index x.

## Output

In mảng sau khi chèn.

## Ví dụ

### Input

```text
4
1 2 4 5
2 3
```

### Output

```text
1 2 3 4 5
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
