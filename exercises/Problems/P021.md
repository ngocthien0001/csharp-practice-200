# P021 - Tiền điện đơn giản

**Mức độ:** Cơ bản  
**Chủ đề:** Điều kiện  
**Tags:** Điều kiện

---

## Đề bài

Tính tiền điện theo bậc: 0-50 kWh giá 1800, phần trên 50 kWh giá 2500.

## Input

Một số nguyên kWh.

## Output

In tổng tiền điện.

## Ví dụ

### Input

```text
40
```

### Output

```text
72000
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
