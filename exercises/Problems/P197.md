# P197 - Số dư theo tài khoản

**Mức độ:** Trung bình  
**Chủ đề:** LINQ/Data  
**Tags:** LINQ/Data

---

## Đề bài

Nhập các giao dịch gồm account và amount. Tính tổng amount theo account.

## Input

Dòng 1: n. n dòng: account amount.

## Output

Mỗi dòng: account balance, sắp xếp account.

## Ví dụ

### Input

```text
4
a 100
b 50
a -20
c 0
```

### Output

```text
a 80
b 50
c 0
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
