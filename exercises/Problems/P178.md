# P178 - Class Person - sắp xếp theo tuổi

**Mức độ:** Trung bình  
**Chủ đề:** OOP  
**Tags:** OOP

---

## Đề bài

Tạo class Person gồm Name, Age. Sắp xếp tăng dần theo tuổi, nếu bằng tuổi sắp xếp theo tên.

## Input

Dòng 1: n. n dòng: name age.

## Output

In danh sách sau sắp xếp.

## Ví dụ

### Input

```text
3
An 20
Binh 18
Chi 20
```

### Output

```text
Binh 18
An 20
Chi 20
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
