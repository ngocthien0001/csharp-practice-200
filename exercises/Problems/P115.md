# P115 - Tần suất 0 đến 100

**Mức độ:** Cơ bản  
**Chủ đề:** Mảng/List  
**Tags:** Mảng/List

---

## Đề bài

Nhập n và mảng gồm các số từ 0 đến 100. In các giá trị xuất hiện kèm số lần xuất hiện, tăng dần theo giá trị.

## Input

Dòng 1: n. Dòng 2: n số nguyên.

## Output

Mỗi dòng: value count.

## Ví dụ

### Input

```text
6
1 2 1 3 2 1
```

### Output

```text
1 3
2 2
3 1
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
