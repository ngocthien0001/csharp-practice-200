# P103 - Dãy con tăng liên tiếp dài nhất

**Mức độ:** Trung bình  
**Chủ đề:** Mảng/List  
**Tags:** Mảng/List

---

## Đề bài

Nhập n và mảng. Tìm độ dài dãy con liên tiếp tăng nghiêm ngặt dài nhất.

## Input

Dòng 1: n. Dòng 2: n số.

## Output

In độ dài lớn nhất.

## Ví dụ

### Input

```text
6
1 2 3 1 2 5
```

### Output

```text
3
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
