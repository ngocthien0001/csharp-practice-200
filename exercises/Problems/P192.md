# P192 - Lọc sản phẩm theo giá

**Mức độ:** Trung bình  
**Chủ đề:** LINQ/Data  
**Tags:** LINQ/Data

---

## Đề bài

Nhập danh sách sản phẩm và khoảng giá [min,max]. In tên sản phẩm nằm trong khoảng giá.

## Input

Dòng 1: n. Dòng 2: min max. n dòng: name price.

## Output

In tên sản phẩm hoặc None.

## Ví dụ

### Input

```text
3
100 500
A 50
B 200
C 500
```

### Output

```text
B C
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
