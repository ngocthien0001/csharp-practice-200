# P149 - Đệ quy Fibonacci

**Mức độ:** Trung bình  
**Chủ đề:** Đệ quy  
**Tags:** Đệ quy

---

## Đề bài

Nhập n. Tính Fn với F0=0, F1=1.

## Input

Một số nguyên n từ 0 đến 35.

## Output

In Fn.

## Ví dụ

### Input

```text
7
```

### Output

```text
13
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
