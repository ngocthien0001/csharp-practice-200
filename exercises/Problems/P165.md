# P165 - Độ dài LCS

**Mức độ:** Khá  
**Chủ đề:** Quy hoạch động  
**Tags:** Quy hoạch động

---

## Đề bài

Nhập hai chuỗi. Tìm độ dài dãy con chung dài nhất.

## Input

Hai dòng chuỗi.

## Output

In độ dài LCS.

## Ví dụ

### Input

```text
abcde
ace
```

### Output

```text
3
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
