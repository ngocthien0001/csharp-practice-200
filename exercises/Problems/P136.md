# P136 - Lấy username từ email

**Mức độ:** Cơ bản  
**Chủ đề:** Chuỗi  
**Tags:** Chuỗi

---

## Đề bài

Nhập email. In phần trước ký tự @.

## Input

Một dòng email.

## Output

In username.

## Ví dụ

### Input

```text
user@example.com
```

### Output

```text
user
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
