# P121 - Đếm số từ

**Mức độ:** Cơ bản  
**Chủ đề:** Chuỗi  
**Tags:** Chuỗi

---

## Đề bài

Nhập một dòng văn bản. Đếm số từ, các từ cách nhau bởi một hoặc nhiều khoảng trắng.

## Input

Một dòng văn bản.

## Output

In số từ.

## Ví dụ

### Input

```text
hello world
```

### Output

```text
2
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
