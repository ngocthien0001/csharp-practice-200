# P094 - Hợp hai mảng

**Mức độ:** Trung bình  
**Chủ đề:** Mảng/List  
**Tags:** Mảng/List

---

## Đề bài

Nhập hai mảng. In hợp các giá trị, không trùng, tăng dần.

## Input

Dòng 1: n. Dòng 2: n số. Dòng 3: m. Dòng 4: m số.

## Output

In hợp của hai mảng.

## Ví dụ

### Input

```text
3
1 2 3
3
3 4 5
```

### Output

```text
1 2 3 4 5
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
