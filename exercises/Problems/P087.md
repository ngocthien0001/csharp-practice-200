# P087 - Kiểm tra mảng tăng dần

**Mức độ:** Cơ bản  
**Chủ đề:** Mảng/List  
**Tags:** Mảng/List

---

## Đề bài

Nhập n và mảng. Kiểm tra mảng có tăng dần không giảm hay không.

## Input

Dòng 1: n. Dòng 2: n số nguyên.

## Output

In YES hoặc NO.

## Ví dụ

### Input

```text
5
1 2 2 3 4
```

### Output

```text
YES
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
