# P065 - Đổi nhị phân sang thập phân

**Mức độ:** Cơ bản  
**Chủ đề:** Vòng lặp  
**Tags:** Vòng lặp

---

## Đề bài

Nhập chuỗi nhị phân. Đổi sang số thập phân.

## Input

Một chuỗi chỉ gồm 0 và 1.

## Output

In giá trị thập phân.

## Ví dụ

### Input

```text
1010
```

### Output

```text
10
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
