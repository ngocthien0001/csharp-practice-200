# P163 - Đổi tiền ít đồng nhất

**Mức độ:** Khá  
**Chủ đề:** Quy hoạch động  
**Tags:** Quy hoạch động

---

## Đề bài

Cho n loại mệnh giá và số tiền amount. Tìm số đồng ít nhất để tạo ra amount, mỗi loại dùng không giới hạn.

## Input

Dòng 1: n amount. Dòng 2: n mệnh giá.

## Output

In số đồng ít nhất hoặc -1.

## Ví dụ

### Input

```text
3 11
1 2 5
```

### Output

```text
3
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
