# P145 - Ký tự xuất hiện nhiều nhất

**Mức độ:** Trung bình  
**Chủ đề:** Chuỗi  
**Tags:** Chuỗi

---

## Đề bài

Nhập một chuỗi. Tìm ký tự xuất hiện nhiều nhất. Nếu hòa, lấy ký tự xuất hiện trước.

## Input

Một dòng chuỗi.

## Output

In ký tự và số lần xuất hiện.

## Ví dụ

### Input

```text
banana
```

### Output

```text
a 3
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
