# P099 - Xóa tất cả giá trị x

**Mức độ:** Trung bình  
**Chủ đề:** Mảng/List  
**Tags:** Mảng/List

---

## Đề bài

Nhập n, mảng và x. Xóa mọi phần tử bằng x. Nếu mảng rỗng sau khi xóa in Empty.

## Input

Dòng 1: n. Dòng 2: n số. Dòng 3: x.

## Output

In mảng còn lại hoặc Empty.

## Ví dụ

### Input

```text
5
1 2 2 3 2
2
```

### Output

```text
1 3
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
