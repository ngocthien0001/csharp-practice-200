# P117 - Đếm nguyên âm

**Mức độ:** Cơ bản  
**Chủ đề:** Chuỗi  
**Tags:** Chuỗi

---

## Đề bài

Nhập một chuỗi. Đếm số nguyên âm a, e, i, o, u, không phân biệt hoa thường.

## Input

Một dòng chuỗi.

## Output

In số nguyên âm.

## Ví dụ

### Input

```text
Hello World
```

### Output

```text
3
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
