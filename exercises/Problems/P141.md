# P141 - Giải nén RLE

**Mức độ:** Trung bình  
**Chủ đề:** Chuỗi  
**Tags:** Chuỗi

---

## Đề bài

Nhập chuỗi dạng ký tự + số lần lặp. Giải nén chuỗi.

## Input

Một dòng chuỗi RLE.

## Output

In chuỗi gốc.

## Ví dụ

### Input

```text
a3b2c1
```

### Output

```text
aaabbc
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
