# P123 - Đảo hoa thường

**Mức độ:** Cơ bản  
**Chủ đề:** Chuỗi  
**Tags:** Chuỗi

---

## Đề bài

Nhập một chuỗi. Đổi chữ hoa thành thường và chữ thường thành hoa. Ký tự khác giữ nguyên.

## Input

Một dòng chuỗi.

## Output

In chuỗi sau khi đổi.

## Ví dụ

### Input

```text
AbC123
```

### Output

```text
aBc123
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
