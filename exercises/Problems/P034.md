# P034 - Góc phần tư

**Mức độ:** Cơ bản  
**Chủ đề:** Điều kiện  
**Tags:** Điều kiện

---

## Đề bài

Nhập tọa độ điểm (x,y). Xác định góc phần tư. Nếu nằm trên trục hoặc gốc tọa độ in Axis.

## Input

Một dòng gồm x và y.

## Output

In Q1, Q2, Q3, Q4 hoặc Axis.

## Ví dụ

### Input

```text
1 2
```

### Output

```text
Q1
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
