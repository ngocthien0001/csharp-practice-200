# P011 - Tổng chữ số số có hai chữ số

**Mức độ:** Cơ bản  
**Chủ đề:** Toán học  
**Tags:** Toán học

---

## Đề bài

Nhập số nguyên n có hai chữ số. In tổng hai chữ số.

## Input

Một số nguyên n từ 10 đến 99.

## Output

In tổng các chữ số.

## Ví dụ

### Input

```text
25
```

### Output

```text
7
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
