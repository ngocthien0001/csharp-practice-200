# P162 - Số đường đi trong lưới

**Mức độ:** Khá  
**Chủ đề:** Quy hoạch động  
**Tags:** Quy hoạch động

---

## Đề bài

Cho lưới n x m gồm . là ô trống và # là vật cản. Từ góc trên trái đi xuống hoặc sang phải đến góc dưới phải. Đếm số đường đi.

## Input

Dòng 1: n m. Sau đó n dòng lưới.

## Output

In số đường đi modulo 1000000007.

## Ví dụ

### Input

```text
3 3
...
.#.
...
```

### Output

```text
2
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
