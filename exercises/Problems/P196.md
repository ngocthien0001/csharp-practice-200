# P196 - Gộp cập nhật gold tài khoản

**Mức độ:** Trung bình  
**Chủ đề:** LINQ/Data  
**Tags:** LINQ/Data

---

## Đề bài

Có danh sách gold ban đầu và danh sách cập nhật cộng/trừ gold. Gộp kết quả theo username.

## Input

Dòng 1: n. n dòng: username gold. Dòng tiếp: m. m dòng: username deltaGold.

## Output

Mỗi dòng: username finalGold, sắp xếp username.

## Ví dụ

### Input

```text
2
acc1 100
acc2 200
3
acc1 50
acc3 300
acc2 -100
```

### Output

```text
acc1 150
acc2 100
acc3 300
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
