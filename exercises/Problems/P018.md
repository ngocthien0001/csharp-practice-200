# P018 - Khoảng cách hai điểm

**Mức độ:** Cơ bản  
**Chủ đề:** Toán học  
**Tags:** Toán học

---

## Đề bài

Nhập tọa độ hai điểm A(x1,y1), B(x2,y2). Tính khoảng cách AB.

## Input

Một dòng gồm x1 y1 x2 y2.

## Output

In khoảng cách với 2 chữ số sau dấu phẩy.

## Ví dụ

### Input

```text
0 0 3 4
```

### Output

```text
5.00
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
