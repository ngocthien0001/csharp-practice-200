# P016 - Chỉ số BMI

**Mức độ:** Cơ bản  
**Chủ đề:** Toán học  
**Tags:** Toán học

---

## Đề bài

Nhập cân nặng kg và chiều cao mét. Tính BMI = weight / height^2.

## Input

Một dòng gồm weight và height.

## Output

In BMI với 2 chữ số sau dấu phẩy.

## Ví dụ

### Input

```text
60 1.70
```

### Output

```text
20.76
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
