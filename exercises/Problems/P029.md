# P029 - Xếp loại điểm

**Mức độ:** Cơ bản  
**Chủ đề:** Điều kiện  
**Tags:** Điều kiện

---

## Đề bài

Nhập điểm từ 0 đến 10. Xếp loại: >=8 Good, >=6.5 Fair, >=5 Average, còn lại Weak.

## Input

Một số thực score.

## Output

In xếp loại.

## Ví dụ

### Input

```text
8.5
```

### Output

```text
Good
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
