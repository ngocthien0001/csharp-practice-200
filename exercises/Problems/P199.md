# P199 - Tọa độ sau các lệnh di chuyển

**Mức độ:** Trung bình  
**Chủ đề:** Mô phỏng  
**Tags:** Mô phỏng

---

## Đề bài

Bắt đầu tại (0,0). Có các lệnh U/D/L/R k. Tính tọa độ cuối.

## Input

Dòng 1: n. n dòng: direction step.

## Output

In x y cuối cùng.

## Ví dụ

### Input

```text
4
U 2
R 3
D 1
L 5
```

### Output

```text
-2 1
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
