# P077 - Trung bình mảng

**Mức độ:** Cơ bản  
**Chủ đề:** Mảng/List  
**Tags:** Mảng/List

---

## Đề bài

Nhập n và mảng n số nguyên. Tính trung bình cộng.

## Input

Dòng 1: n. Dòng 2: n số nguyên.

## Output

In trung bình với 2 chữ số sau dấu phẩy.

## Ví dụ

### Input

```text
4
1 2 3 4
```

### Output

```text
2.50
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
