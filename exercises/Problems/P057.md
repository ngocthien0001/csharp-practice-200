# P057 - In dãy Fibonacci

**Mức độ:** Cơ bản  
**Chủ đề:** Vòng lặp  
**Tags:** Vòng lặp

---

## Đề bài

Nhập n. In n số Fibonacci đầu tiên bắt đầu từ F0.

## Input

Một số nguyên n >= 1.

## Output

In dãy trên một dòng.

## Ví dụ

### Input

```text
6
```

### Output

```text
0 1 1 2 3 5
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
