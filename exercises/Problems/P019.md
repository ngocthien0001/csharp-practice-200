# P019 - Chu vi tam giác

**Mức độ:** Cơ bản  
**Chủ đề:** Toán học  
**Tags:** Toán học

---

## Đề bài

Nhập ba cạnh a, b, c của tam giác. Tính chu vi.

## Input

Một dòng gồm ba số nguyên a, b, c.

## Output

In chu vi.

## Ví dụ

### Input

```text
3 4 5
```

### Output

```text
12
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
