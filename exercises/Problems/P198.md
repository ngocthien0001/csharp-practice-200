# P198 - Parser key=value

**Mức độ:** Trung bình  
**Chủ đề:** Xử lý dữ liệu  
**Tags:** Xử lý dữ liệu

---

## Đề bài

Nhập nhiều dòng key=value. Sau đó nhập key cần tìm, in value tương ứng.

## Input

Dòng 1: n. n dòng: key=value. Dòng cuối: key.

## Output

In value hoặc Not found.

## Ví dụ

### Input

```text
3
name=Thien
role=dev
city=HCM
role
```

### Output

```text
dev
```

## Gợi ý luyện tập

- Đọc input thật kỹ trước khi code.
- Tách biến rõ tên, tránh viết quá dồn 1 dòng.
- Chạy **Ví dụ** trước, sau đó mới bấm **Chấm bài**.
